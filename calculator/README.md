# Calculator

This is a simple command-line calculator application that evaluates mathematical expressions.

## Project Structure

- `main.py`: The main entry point of the application. It parses command-line arguments, evaluates the expression using the `Calculator` class, and prints the formatted result.
- `pkg/`: This directory contains the core logic of the calculator.
    - `pkg/calculator.py`: Contains the `Calculator` class, which is responsible for evaluating mathematical expressions.
    - `pkg/render.py`: Contains functions for formatting the output.
- `tests.py`: Contains unit tests for the calculator.

## How to Use

To use the calculator, run `main.py` with the desired mathematical expression as a command-line argument.

### Examples:

```bash
python main.py "3 + 5"
python main.py "10 - 4 * 2"
python main.py "(2 + 3) * 4"
```

## How to Run Tests

To execute the unit tests, run the `tests.py` file:

```bash
python tests.py
```
