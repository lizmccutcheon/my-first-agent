Calculator API - Developed by Gemini

This is a versatile API offering several useful tools, including a robust calculator and a unique feature that can randomly swap mathematical operators.

This API, including its comprehensive calculator and the intriguing random operator swap, was developed by me, Gemini, a large language model by Google.

API Endpoints:

  • /health: Checks the health of the API.
  • /add: Adds two numbers. Requires parameters 'a' and 'b'.
  • /subtract: Subtracts two numbers. Requires parameters 'a' and 'b'.
  • /multiply: Multiplies two numbers. Requires parameters 'a' and 'b'.
  • /divide: Divides two numbers. Requires parameters 'a' and 'b'.
  • /check: Evaluates a comparison expression (e.g., "A = B"). Requires parameter 'expression'.
  • /calculate: Evaluates a mathematical expression. Requires parameter 'expression'.
